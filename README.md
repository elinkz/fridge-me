# FridgeMe
Recipe generating web application built in React.js with es2015. This is a school project. 

### Version
1.0.0

### Tech
- React.js
- Node.js
- Gulp
- SASS

### Installation

- Clone this repo

- Install Gulp globally:
```sh
$ npm install -g gulp
```
- Change directory:
```sh
$ cd fridge-me
```
- Install packages:
```sh
$ npm install 
```
- Run gulp!
```sh
$ gulp 
```
